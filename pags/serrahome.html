¿Qué es SERRAHOME?

SERRAHOME es un sistema de gestión de reservas para complejos de cabañas, desarrollado como proyecto final de la carrera de Analista Universitario en Sistemas en la Universidad Tecnológica Nacional, Facultad Regional Córdoba. Este software busca optimizar la administración de reservas, proporcionando una solución funcional, escalable e intuitiva para alojamientos situados en las Sierras Cordobesas en primera instancia, aunque da soporte a otros lugares.

Utilizamos tecnologías modernas como Next.js, como framework para integrar el frontend y el backend en el mismo proyecto; React, para construir interfaces de usuario dinámicas y reutilizables; Prisma, como ORM para gestionar la base de datos relacional PostgreSQL; y Docker, para simplificar la implementación en entornos controlados. Además, empleamos Tailwind CSS para estilizar el sistema con un diseño responsive.

El desarrollo se llevó a cabo aplicando la metodología ágil Scrum, lo que nos permitió trabajar de manera iterativa y colaborativa. Organizamos el proyecto en 5 sprints a lo largo de 4 meses. Cada sprint incluyó etapas de planificación, desarrollo y revisión, con ceremonias como reuniones de planificación y revisiones al finalizar cada ciclo. Este enfoque ágil no solo nos ayudó a mantener el foco en las necesidades del cliente, sino también a adaptarnos rápidamente a cambios y desafíos.

SERRAHOME es más que un software; es el reflejo de nuestro esfuerzo, aprendizaje y pasión por la tecnología, un proyecto que representa el cierre de una etapa y el comienzo de nuevas oportunidades.
